# 魔法の数字検索 (Magic Number Search)

Webサイト上で発見した「魔法の数字（作品番号など）」を選択し、右クリックからワンクリックでGoogle検索や「お気に入り」登録ができるChrome拡張機能です。

## 主な機能

- **ワンクリック検索**: Webページ上で数字を選択して右クリックし、専用のクエリで検索を実行できます。
- **お気に入り管理**: あとで調べたい数字を右クリックから「お気に入り」として保存できます。
- **クイックアクセス**: 追加したお気に入りの直近5件は、右クリックメニューからすぐに再検索可能です。
- **CSVエクスポート**: 検索履歴やお気に入りの一覧を、設定画面からいつでもCSV形式で出力できます。
- **プライバシー配慮**: 自動走査や自動判定を行わず、すべてのデータはブラウザのローカルストレージにのみ保存されます。

## インストール方法

Chromeウェブストアを経由せず、手動でインストール（デベロッパーモード）して使用します。

1. このリポジトリのソースコードをダウンロードし、フォルダに展開します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理画面を開きます。
3. 画面右上の **「デベロッパー モード」** のスイッチをオンにします。
4. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5. 展開したフォルダを選択すると、インストールが完了します。

## 使い方

1. 任意のWebページ上で、調べたい数字や文字列をマウスでドラッグして選択します。
2. 選択した状態で **右クリック** します。
3. メニューから「◆ 魔法の数字」にカーソルを合わせます。
4. 以下の操作が選べます：
   - `検索する`: 選択した数字を使ってGoogle検索を新しいタブで開きます。
   - `★ お気に入りに追加/削除`: 数字をお気に入りに登録（または解除）します。
   - `・ [数字] を検索`: 過去に追加したお気に入りから素早く検索します。
5. メニュー内の「■ お気に入り・履歴の管理」をクリックすると、管理画面が開き、データの確認やCSVダウンロードが可能です。
